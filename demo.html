Target.js Demo

Target.js doesn't do much; but what it does, it does well. It adds functionality to your DOM elements using data-attributes.

View Demo View on GitHub Download .zip Download .tar.gz

Basic Component Usage

Show

Show an element

<button data-target-show="#show-demo-div">
  Show Div Below
</button>

<div id="show-demo-div">
</div>

Hide

Hide an element

<button data-target-hide="#hide-demo-div">
  Hide Div Below
</button>

<div id="hide-demo-div">
</div>

Toggle

Toggle an element

<button data-target-toggle="#toggle-demo-div">
  Toggle Div Below
</button>

<div id="toggle-demo-div">
</div>

Clickoff

Click off of an element to hide

<button data-target-show="#clickoff-demo-div">
  Show div
</button>

<div id="clickoff-demo-div" data-target-clickoff>
  <p>Click away from div to hide</p>
</div>

Click away from div to hide

Increment/Decrement

Increment and Decrement the value of an input

<button data-target-decrement="#qty">-</button>
<input id="qty" value="1">
<button data-target-increment="#qty" data-target-max="10">+</button>

Scrollbox

Add scrollbars to element when height over threshold

<div data-target-scrollbox="300">
        Put any HTML here.
</div>
